1) User Management
2) Login wrt to Role
3) Job Creation
4) Job Status Update
5) Job Image Upload
6) Job Completion
7) Job Auditing
8) Job DA/TA/PO calculation and updation
9) Create Site
10) Create Circles,Operators,Activities and Projects
